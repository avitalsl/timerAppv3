# NewTimerApp

A timer application for managing meetings and activities.

## Project Structure

- `/newStandApp` - Main application code

## Development

### Prerequisites

- Node.js
- npm

### Setup

1. Clone the repository
2. Install dependencies:
```bash
cd newStandApp
npm install
```

3. Start the development server:
```bash
npm run dev
```

## Contributing
